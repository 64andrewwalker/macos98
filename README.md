# macOS 98

A late-90s Mac OS desktop simulator built with React 19, TypeScript, Vite, and SCSS modules.

## Quick Start

```bash
pnpm install
pnpm dev
```

See [Getting Started](docs/getting-started.md) for full setup instructions.
